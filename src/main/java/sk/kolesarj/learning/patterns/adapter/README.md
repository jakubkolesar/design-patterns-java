# Adapter
Predstavme si, že máme elektrické zariadenie. V Európe sú ine zástrčky ako v Amerike
a zariadenia taktiež podporujú inú "voltáž".

Adapter je vzor, ktorý dovoľuje objektom s nekompatibilnými interface-ami 
kolaborovať.
